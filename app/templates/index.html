Pet Demo REST API Service

GET /pets to see all the pets.

GET /categories to see all the categories.

GET /pets?category=dog to see all of the pets in the 'dog' category.

GET /pets/{id} retrieves a pet from the DB using an ID.

PUT /pets/{id} updates a pet using an ID.

DELETE /pets/{id} removes a pet from the DB using an ID.

POST /pets creates an new pet.

Example Data:

    {
        "name": "bowser",
        "category": 1,
        "available": true
    }
